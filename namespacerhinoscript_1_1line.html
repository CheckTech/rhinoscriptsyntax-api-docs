Calculates the intersection of a line and a cylinder.

Parameters
line = the line to intersect cylinder_plane = base plane of the cylinder cylinder_height = height of the cylinder cylinder_radius = radius of the cylinder
Returns
Determines if the shortest distance from a line to a point or another line is greater than a specified distance
True if the shortest distance from the line to the other project is greater than distance, False otherwise

list of intersection points (0, 1, or 2 points)
