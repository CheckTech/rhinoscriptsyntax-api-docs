# SurfaceDegree

Returns the degree of a surface object in the specified direction
        

### Parameters:

- ***surface_id*** = the surface's identifier
direction[opt]
  0 = U, 1 = v, 2 = both
        

### Returns:


Tuple of two values if direction = 2
Single number if direction = 0 or 1
None on error
