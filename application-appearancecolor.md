# AppearanceColor

Returns or modifies an application interface item's color.
          

### Parameters:

- ***item*** = Item number to either query or modify
       0  = View background
       1  = Major grid line
       2  = Minor grid line
       3  = X-Axis line
       4  = Y-Axis line
       5  = Selected Objects
       6  = Locked Objects
       7  = New layers
       8  = Feedback
       9  = Tracking
       10 = Crosshair
       11 = Text
       12 = Text Background
       13 = Text hover
- ***color[opt]*** = The new color value
        

### Returns:


if color is not specified, the current item color
if color is specified, the previous item color
