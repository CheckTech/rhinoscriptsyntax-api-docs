DimStyleOffset

Returns or changes the extension line offset of a dimension style.

Syntax

rhinoscriptsyntax.DimStyleOffset (dimstyle, offset=None )

rhinoscript.dimension.DimStyleOffset (dimstyle, offset=None )

Parameters

dimstyle

Required.  String.  The name of an existing dimension style.

offset

Optional.  Number.  The new extension line offset.  If omitted, the current extension line offset is returned.

Returns

Number

If offset is not specified, the current extension line offset if successful.

Number

If offset is specified, the previous extension line offset if successful.

None

If not successful, or on error.

Example

import rhinoscriptsyntax as rs

dimstyle = rs.CurrentDimStyle()

offset = rs.DimStyleOffset(dimstyle)

if offset>0.5: rs.DimStyleOffset( dimstyle, 0.5 )
