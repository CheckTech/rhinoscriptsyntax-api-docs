PointArrayBoundingBox

Returns either a world axis-aligned or a construction plane axis-aligned bounding box of an array of 3-D point locations.

Syntax

rhinoscriptsyntax.PointArrayBoundingBox (points, view_or_plane=None, in_world_coords=True)

rhinoscript.pointvector.PointArrayBoundingBox (points, view_or_plane=None, in_world_coords=True)

 

Parameters

points

Required.  List.  A list of 3-D points.

view_or_plane

Optional.  String.  Title or id of the view that contains the construction plane to which the bounding box should be aligned -or- user defined plane. If omitted, a world axis-aligned bounding box will be calculated.

in_world_coords

Optional.  Boolean.  Whether or not to return the bounding box as world coordinates or construction plane coordinates.  The default is to return world coordinates (True).  Note, this option does not apply to world axis-aligned bounding boxes.

Returns

List

A list of eight 3-D points that define the bounding box if successful.  Points are returned in counter-clockwise order starting with the bottom rectangle of the box.  See the image below for details.

None

If not successful, or on error.
