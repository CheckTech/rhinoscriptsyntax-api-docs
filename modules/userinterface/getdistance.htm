GetDistance

Pauses for user input of a distance.

Syntax

rhinoscriptsyntax.GetDistance (first_pt=None, distance=None, first_pt_msg='First distance point', second_pt_msg='Second distance point')

Parameters

first_pt

Optional.  First distance point.

distance

Optional.  Number.  A default distance value.

first_pt_msg

Optional.  String.  A prompt or message for the first distance point.

second_pt_msg

Optional.  String.  A prompt or message for the second distance point.

Returns

Number

The distance input by the user if successful.

None

If not successful, or on error.

Example

import rhinoscriptsyntax as rs

dist = rs.GetDistance()

if dist:

    print dist
