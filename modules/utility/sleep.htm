Sleep

Suspends the execution of a running script for the specified interval.

Syntax

rhinoscriptsyntax.Sleep (milliseconds)

rhinoscript.utility.Sleep (milliseconds)

Parameters

milliseconds

Required.  Number.  The duration in milliseconds.

Returns

None

If successful, or on error.

Example

import rhinoscriptsyntax as rs

print "This"

rs.Sleep(2000)

print "is"

rs.Sleep(2000)

print "a"

rs.Sleep(2000)

print "slow"

rs.Sleep(2000)

print "message!"
