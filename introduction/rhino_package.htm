The rhinoscriptsyntax reference

Importing the rhinoscriptsyntax package will import all modules.  For example, adding:

import rhinoscriptsyntax as rs

at the top of your script gives you access to all functions in all modules.  For example to call the AddCircle(center, radius) function from the curve module just type:

rs.AddCircle((0,0,0), 25.0)

The following table describes each module and the link will bring you to a description of all the available functions.

 

Module

Description

application

Functions related to the Rhino application.

block

Functions related to block instances and their definitions.

curve

Functions related to curve objects.

dimension

Functions related to dimension and annotations.

document

Functions related to the currently open 3DM file.

geometry

Functions related to geometric objects.

grips

Functions for manipulating object grips.

group

Functions used to manipulate groups of objects.

hatch

Functions used to manipulate hatch objects.

layer

Functions used to manipulate layers.

light

Functions related to light objects.

line

Functions related to line objects.

linetype

Functions related to linetype objects.

material

Functions related to object rendering materials.

mesh

Functions related to mesh objects.

object

Functions to manipulate general object properties.

plane

Functions related to manipulating 3-D lines and planes.

pointvector

Functions related to creating and manipulating 3-D points and 3-D vectors.

selection

Functions for selecting objects or obtaining object identifiers.

surface

Functions related to surface and polysurface objects.

toolbar

Functions related to toolbar collections.

transformation

Functions related to creating transformation.

userdata

Functions related to userdata.

userinterface

Functions used to interact with the user.

utility

Any function that does not fall into any of the above categories.

view

Functions related to manipulate views and construction planes.
