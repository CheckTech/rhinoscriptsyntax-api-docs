def rhinoscript.block.BlockCount ( )

Returns or sets the description of a block definition.

Parameters
block_name = the name of an existing block definition description[opt] = The new description.
Returns
Counts number of instances of the block in the document. Nested instances are not included in the count.
Parameters
block_name = the name of an existing block definition where_to_look [opt] = 0 = get top level references in active document. 1 = get top level and nested references in active document. 2 = check for references from other instance definitions

if description is not specified, the current description if description is specified, the previous description
