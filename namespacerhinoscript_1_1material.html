def rhinoscript.material.AddMaterialToLayer (   layer)

Add material to a layer and returns the new material's index.

If the layer already has a material, then the layer's current material index is returned

Parameters
layer = name of an existing layer.
Returns
Material index of the layer if successful None if not successful or on error

def rhinoscript.material.IsMaterialDefault (   material_index)

Verifies a material is a copy of Rhino's built-in "default" material.

The default material is used by objects and layers that have not been assigned a material.

Parameters
material_index = the zero-based material index
Returns
True or False indicating success or failure
